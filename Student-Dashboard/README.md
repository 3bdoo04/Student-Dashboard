# Student Dashboard

A full-stack web application for managing student records.

## Features
- Add new students with name, email, and course
- View all students in a clean list
- Data stored in JSON file
- Simple, clean UI

## Setup

### 1. Prerequisites
- Node.js installed on your computer
- Web browser

### 2. Installation

**Backend Setup:**

# Open terminal in the project folder
- cd backend
- npm install
- node server.js
